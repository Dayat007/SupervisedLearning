#Install Streamlit
1. Install streamlit dengan CMD "pip install streamlit"
2. Check apakah streamlit terinstall dengan command "streamlit hello"

#Pada folder sudah tersedia Model Supervised Learning
3. Buat model Supervised Learning (Classification & Regression) dengan menggunakan python
4. Letakkan file model pada satu folder yang sama

#Pada folder sudah tersedia program index untuk web (Kode Editor : Visual Studio Code)
5. Buat program python untuk desain website 
6. Jalankan program pada terminal dengan command "streamlit run index.py"
